KiCad import bundle
===================

This archive contains only two folders:
1) KiCad_Symbols_All: all .kicad_sym symbol libraries
2) KiCad_Footprints_All: all .pretty footprint libraries

Add .kicad_sym files via Preferences > Manage Symbol Libraries.
Add .pretty folders via Preferences > Manage Footprint Libraries.
Then use Tools > Assign Footprints and Tools > Update PCB from Schematic.
